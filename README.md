## 🌌 ¿Qué es Starfall?

**Starfall** es un sistema de partículas minimalista desarrollado para la terminal. Diseñado con el principio de *performance per watt*, busca ofrecer una experiencia visual elegante y fluida sin consumir recursos innecesarios (bloatware). 

A diferencia de otros visualizadores, **Starfall** implementa un sistema de física diagonal para las partículas, creando un efecto de profundidad y movimiento dinámico que transforma cualquier entorno de trabajo en una ventana al espacio profundo.
